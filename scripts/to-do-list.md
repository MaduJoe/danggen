- [ ] **CommunityPage 클래스 구현**
  - 동네생활 탭 네비게이션
  - 게시글 작성 폼 요소
  - 댓글 작성 및 조회 기능
  
- [ ] **커뮤니티 API 클라이언트 확장**
  - 게시글 CRUD API
  - 댓글 관리 API
  - 동네별 게시글 필터링

- [ ] **E2E 테스트 시나리오 구현**
  ```python
  def test_community_engagement_flow():
      # 1. 특정 동네 위치의 사용자 생성
      # 2. 동네생활 탭에서 새 게시글 작성
  ```

- 댓글 작성 후 실시간으로 반영되는지 검증
- 다른 동네 사용자에게는 게시글이 보이지 않는지 확인
